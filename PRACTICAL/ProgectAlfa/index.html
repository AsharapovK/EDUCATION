CRM dashboard Дашборды Настройки Проекты О нас

Информация:

Проект разработан в рамках курса HTML/CSS. Он позволяет извлекать данные из Битрикс24 и создавать на их основе информативные дашборды.

Возможности:

Создание дашбордов на основе данных по сделкам, лидам, компаниям и прочее

Технологии:

На данном этапе проект находится в разработке и использует технологии: Apps script, Java Script и др.

Всего подключено:

3 шт.

Данных выгружено:

150000 записей
